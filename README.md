# FOHS ScoreTracker

## Installing

```sh
git clone https://github.com/fohs-score-tracker/fohs-score-tracker.github.io/
cd fohs-score-tracker.github.io
yarn install --dev
```

### Running (production)

```sh
yarn build
# the app is now in dist/
```

### Running (development)

```sh
yarn dev
# open `localhost:3000` in your browser
```
